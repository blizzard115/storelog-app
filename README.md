# TechLog

## 📝 概要
StoreLogは、アパレル店舗における情報共有を効率化するためのWebアプリケーションです。

従来の紙のノートによる情報共有をデジタル化し、
「誰が読んだか」「誰が読んでいないか」を可視化することで、伝達漏れを防ぎます。

---

## 🎯 作成背景
アパレル店舗での勤務経験の中で、以下の課題を感じました。

- 情報共有が紙のノートで管理されている
- 読んだかどうかの確認が手書きで曖昧
- 忙しい時間帯に情報が埋もれてしまう
- 入荷情報や注意事項がリアルタイムで共有されない

これらの課題を解決するために、
情報共有と既読管理をデジタル化したアプリとしてStoreLogを開発しました。

---

## ✨ アプリの特徴

- 店舗向け情報共有ツール
- 既読管理により伝達漏れを防止
- 未読者を可視化
- シンプルなUIで現場でも使いやすい設計

---

## ⚙️ 主な機能

### 📢 投稿機能（連絡・共有）
- 店舗内の情報（入荷・注意事項など）を投稿
- タイトル・本文の入力

### 👀 既読機能
- 学習記録とエラーログを分類

### 🚨 未読管理
- 未読ユーザーの一覧表示
- 確認漏れを防止

### 🧥 入荷情報管理（予定）
- 新商品の入荷情報を共有

### 👤 ユーザー管理
- ユーザーごとの投稿・既読管理

---

## 🛠 使用技術

- Tailwind CSS
